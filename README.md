## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/azkaabid/zedsoft-project.git
cd zedsoft-project
```

### 2. Install Dependencies:
```bash
npm install
```

3. Start Development Server:
```bash
npm run dev
```


App will be available at http://localhost:5173 (or Vite default port).
